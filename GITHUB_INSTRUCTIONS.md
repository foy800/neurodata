# Инструкции по отправке проекта на GitHub

## 1. Создайте репозиторий на GitHub

1. Перейдите на [GitHub](https://github.com)
2. Войдите в свой аккаунт
3. Нажмите на "+" в правом верхнем углу и выберите "New repository"
4. Введите имя репозитория: `neurodatabase`
5. Оставьте репозиторий публичным или сделайте его приватным
6. Не инициализируйте репозиторий с README, .gitignore или лицензией
7. Нажмите "Create repository"

## 2. Отправьте код в репозиторий

После создания репозитория выполните следующие команды в терминале:

```bash
# Добавьте удаленный репозиторий
git remote add origin https://github.com/ВАШЕ_ИМЯ_ПОЛЬЗОВАТЕЛЯ/neurodatabase.git

# Отправьте код в репозиторий
git push -u origin master
```

Замените `ВАШЕ_ИМЯ_ПОЛЬЗОВАТЕЛЯ` на ваше имя пользователя GitHub.

## 3. Проверьте репозиторий

После успешной отправки перейдите на страницу вашего репозитория:

```
https://github.com/ВАШЕ_ИМЯ_ПОЛЬЗОВАТЕЛЯ/neurodatabase
```

Вы должны увидеть все файлы проекта в репозитории.

## 4. Настройка GitHub Pages (опционально)

Если вы хотите опубликовать статическую версию сайта:

1. Перейдите в настройки репозитория (Settings)
2. Прокрутите вниз до раздела "GitHub Pages"
3. В разделе "Source" выберите ветку "master"
4. Нажмите "Save"

После этого ваш сайт будет доступен по адресу:
```
https://ВАШЕ_ИМЯ_ПОЛЬЗОВАТЕЛЯ.github.io/neurodatabase
```

## 5. Деплой на Vercel

После отправки кода на GitHub вы можете легко развернуть проект на Vercel:

1. Перейдите на [Vercel](https://vercel.com)
2. Войдите в свой аккаунт или создайте новый
3. Нажмите "New Project"
4. Импортируйте репозиторий с GitHub
5. Настройте проект (обычно настройки по умолчанию подходят)
6. Нажмите "Deploy"

После деплоя Vercel предоставит вам URL для доступа к вашему приложению.
