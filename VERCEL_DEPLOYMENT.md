# Инструкции по деплою на Vercel

## 1. Создайте аккаунт на Vercel

1. Перейдите на [Vercel](https://vercel.com)
2. Нажмите "Sign Up" и создайте аккаунт
3. Рекомендуется использовать GitHub для регистрации, чтобы легко импортировать репозитории

## 2. Создайте новый проект

1. После входа в аккаунт нажмите "New Project"
2. Выберите опцию "Import Git Repository"
3. Подключите свой аккаунт GitHub, если еще не сделали этого
4. Выберите репозиторий `neurodatabase` из списка

## 3. Настройте проект

1. В разделе "Configure Project" оставьте настройки по умолчанию:
   - Framework Preset: `Other`
   - Root Directory: `./` (корневая директория проекта)
   - Build Command: оставьте пустым
   - Output Directory: оставьте пустым
   - Install Command: `npm install`

2. В разделе "Environment Variables" можно добавить переменные окружения, если они нужны для вашего проекта (в нашем случае не требуется)

## 4. Деплой проекта

1. Нажмите кнопку "Deploy"
2. Дождитесь завершения процесса деплоя
3. После успешного деплоя вы получите URL вашего приложения (например, `https://neurodatabase.vercel.app`)

## 5. Проверьте деплой

1. Перейдите по предоставленному URL
2. Убедитесь, что все страницы и функции работают корректно
3. Проверьте адаптивность дизайна на разных устройствах

## 6. Настройка домена (опционально)

Если у вас есть собственный домен, вы можете настроить его для вашего проекта:

1. В панели управления проектом перейдите в раздел "Settings" -> "Domains"
2. Нажмите "Add" и введите ваш домен
3. Следуйте инструкциям для настройки DNS

## 7. Автоматический деплой

Vercel автоматически настраивает CI/CD для вашего проекта:

1. При каждом push в ветку `master` вашего GitHub репозитория Vercel автоматически запустит новый деплой
2. Вы можете просматривать историю деплоев в разделе "Deployments"

## 8. Мониторинг и аналитика

Vercel предоставляет базовые инструменты для мониторинга:

1. В панели управления проектом перейдите в раздел "Analytics"
2. Здесь вы можете отслеживать посещения, производительность и другие метрики

## Важные замечания

- Vercel оптимизирован для работы с JavaScript-фреймворками, но также хорошо работает с Node.js приложениями
- Бесплатный план Vercel имеет ограничения по количеству деплоев и использованию ресурсов
- Для продакшн-приложений с высокой нагрузкой рекомендуется использовать платные планы
