## Важный шаг: Настройка Vercel

Чтобы сайт заработал в интернете, вам нужно добавить переменные окружения в Vercel.

1. Перейдите в настройки вашего проекта на [Vercel Dashboard](https://vercel.com/dashboard).
2. Выберите ваш проект -> **Settings** -> **Environment Variables**.
3. Добавьте следующие переменные (скопируйте их из вашего локального файла `.env`):

   - **TURSO_DATABASE_URL**
     - Значение: `libsql://neurodata-pavelpro.aws-eu-west-1.turso.io`
   
   - **TURSO_AUTH_TOKEN**
     - Значение: *(Ваш длинный токен, который вы прислали)*
   
   - **SESSION_SECRET**
     - Значение: `neuro-knowledge-base-secret-key-2024-production` (или придумайте свой сложный пароль)
   
   - **NODE_ENV**
     - Значение: `production`

4. После добавления переменных, **передеплойте проект** (Redeploy) в Vercel, чтобы изменения вступили в силу.

## Проверка работы

После деплоя вы можете проверить работу базы данных, открыв страницу:
