# База знаний по нейросетям

Веб-приложение для управления базой знаний с материалами по изучению нейросетей.

## Особенности

- **Аутентификация пользователей**: Регистрация и вход по логину и паролю
- **Административная панель**: Скрытая страница для администратора
- **Управление пользователями**: Возможность блокировки/разблокировки пользователей
- **Категории знаний**: Организация материалов по тематическим разделам
- **Система чата**: Возможность общения между пользователями
- **Профиль пользователя**: Настройка аватара и личной информации
- **Адаптивный дизайн**: Современный интерфейс для всех устройств

## Категории базы знаний

1. **Скрипты продаж** - Готовые скрипты для продаж нейросетевых решений
2. **Промпты для агентов** - Эффективные промпты для создания ИИ-агентов
3. **Промпты для голосовых ассистентов** - Промпты для настройки голосовых помощников
4. **Шаблон сайта** - Готовые шаблоны сайтов с нейросетевыми технологиями
5. **Договора** - Шаблоны договоров для нейросетевых проектов
6. **Коммерческие предложения** - Шаблоны коммерческих предложений

## Установка и запуск

### Требования
- Node.js 14.x или выше
- npm или yarn

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/yourusername/neurodatabase.git
cd neurodatabase

# Установка зависимостей
npm install
# или
yarn install
```

### Запуск

```bash
# Запуск сервера разработки
npm start
# или
yarn start
```

Приложение будет доступно по адресу: `http://localhost:8080`

Проект настроен для деплоя на Vercel. Для деплоя выполните:

```bash

npm install -g vercel

# Деплой
vercel
```
