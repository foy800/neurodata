# Инструкции по созданию репозитория на GitHub

## 1. Создайте репозиторий на GitHub

1. Перейдите на [GitHub](https://github.com)
2. Войдите в свой аккаунт
3. Нажмите на "+" в правом верхнем углу и выберите "New repository"
4. Введите имя репозитория: `neurodatabase`
5. Добавьте описание: "База знаний по нейросетям"
6. Выберите "Public" (публичный репозиторий)
7. НЕ инициализируйте репозиторий с README, .gitignore или лицензией
8. Нажмите "Create repository"

## 2. Отправьте код в репозиторий

После создания репозитория вы увидите инструкции. Выполните следующие команды в терминале:

```bash
# Добавьте удаленный репозиторий (замените YOUR_USERNAME на ваше имя пользователя GitHub)
git remote add origin https://github.com/YOUR_USERNAME/neurodatabase.git

# Отправьте код в репозиторий
git push -u origin master
```

## 3. Проверьте репозиторий

После успешной отправки перейдите на страницу вашего репозитория:

```
https://github.com/YOUR_USERNAME/neurodatabase
```

Вы должны увидеть все файлы проекта в репозитории.

## 4. Настройка GitHub Pages (опционально)

Если вы хотите опубликовать статическую версию сайта:

1. Перейдите в настройки репозитория (Settings)
2. Прокрутите вниз до раздела "GitHub Pages"
3. В разделе "Source" выберите ветку "master"
4. Нажмите "Save"

После этого ваш сайт будет доступен по адресу:
```
https://YOUR_USERNAME.github.io/neurodatabase
```
